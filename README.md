# ThreatSim

An interactive cybersecurity awareness and psychology-based learning simulator built using HTML, CSS, and JavaScript.

ThreatSim teaches users through immersive walkthrough scenarios instead of traditional quizzes. Players experience realistic cybercrime situations inspired by scam trends reported across different Indian states in 2025 and learn how manipulation, urgency, fear, and social engineering are used in real attacks.

---

# 🌐 Live Demo

https://threat-sim-alpha.vercel.app/

# 🚀 Features

- 10 interactive cybercrime simulation levels
- Walkthrough-style learning experience
- Psychology + cybersecurity integrated explanations
- AI cyber mentor system: Sentinel
- Security score and threat meter system
- Level progression and gamified flow
- Reality Check section with real-world scam references
- Responsive mobile-first UI
- Collapsible sidebar navigation
- LocalStorage-based profile and progress saving
- Cyberpunk dark mode interface
- Interactive scam messages, fake portals, calls, and documents

# 🎮 Interactive Learning Approach

ThreatSim focuses on:

- experiential learning
- decision-based interaction
- behavioral awareness
- psychology-driven cybersecurity education

Instead of asking direct MCQs, users:

- interact with scam scenarios
- make decisions under pressure
- observe consequences
- receive contextual explanations

This creates a more immersive and memorable learning experience.

---

# 🤖 Sentinel AI Mentor

Sentinel is an integrated cyber mentor system that:

- reacts to user decisions
- explains scam psychology
- provides cybersecurity insights
- guides players throughout simulations

The mentor dynamically responds based on:

- user behavior
- threat level
- scam type
- risky decisions
